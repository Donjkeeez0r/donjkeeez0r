## 👋 Обо мне

Привет!  **back-end / full-stack разработчик**.

- 🏢 Сейчас прохожу стажировку в крупной телеком-компании — разрабатываю систему автоматизации анкетирования по информационной безопасности для подрядчиков
- 🛠️ Стек стажировочного проекта: **NestJS + Prisma + PostgreSQL** на бэкенде, **React** на фронтенде, JWT-авторизация с ролями
- 🎮 В свободное время делаю пет-проекты: игру в шашки с обучением с подкреплением (Q-learning / DQN), трекер продуктивности, генератор QR-кодов с аналитикой
- 📚 Учусь строить чистую архитектуру, работать с TypeScript "по-взрослому" и постепенно ухожу от учебного кода к продовым практикам
- 🌱 Предпочитаю разбираться в задачах сам — люблю получать подсказки и скелеты кода, а не готовые решения
